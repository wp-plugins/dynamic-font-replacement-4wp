== Description ==

Use your own ttf- or otf-fonts with wordpress. Seo-friendly, fast and easy. 
The plugin does display the document and replace the text in the background. 
The actual source code does remain the same. 
This way search engines won’t have any problems reading and indexing. 
The conversion is quick and easy. You won’t need to implement any extra CSS 
files because the plug-in does detect the tags used and replaces them automatically.

== Installation ==

1. Upload `dfr4wp` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate "Dynamic Font Replacement" in the Admin-menue.
4. Allocate the fonts to the CSS- or HTML-Tags.  
5. See your site.
